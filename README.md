# 🎮 Unoficial Acer Nitro RGB Control (Linux GUI)
A simple Python GUI application to control **RGB keyboard lighting** on Acer Predator/Nitro laptops running Linux.

## 🙌 Credits
This project is built on top of the excellent work by:

Jafar Akhondali
🔗 https://github.com/JafarAkhondali/acer-predator-turbo-and-rgb-keyboard-linux-module


All low-level hardware functionality is powered by this module.

This project provides a user-friendly interface built on top of an existing kernel module, making it easier to control hardware features without using the terminal.

This GUI depends on the kernel module developed by:

👉 https://github.com/JafarAkhondali/acer-predator-turbo-and-rgb-keyboard-linux-module

Follow the setup instructions in that repository to install and load the module before using this application.

## ✨ Features

- 🌈 Control **RGB keyboard lighting**
- 🖥️ Clean and minimal GUI (GTK-based)

## 🧩 Requirements

- Linux (tested on Arch / Hyprland setups recommended)
- Python 3.8+
- GTK (PyGObject)

## ⚠️ Disclaimer
- This application interacts with low-level system components.
- Use at your own risk.
- Not all Acer devices are supported.
